## Layouts

      <td>Reduce clutter inside app folder</td>

    </tr>
    <tr>
      <td rowSpan="2">Store project files in top-level folders in side of app</td>

      <td>Simpicity in routing and file discovery</td>

### Store project files outside of app

```diff
PROJECT
|- app/
|  |- dashboard/
|  |  `- page.tsx
|  `- page.tsx
|- components/
|- lib/
|- ...
`- package.json
```

### Store project files in top-level folders inside of app

```diff
PROJECT
|- app/
|  |- dashboard/
|  |  `- page.tsx
|  |- page.tsx
|  |- components/
|  |- lib/
|- ...
`- package.json
```

### Split project files by feature or route

```diff
PROJECT
|- app/
|  |- components/
|  |- lib/
|  |- dashboard/
|  |  |- components/
|  |  |- lib/
|  |  `- page.tsx
|  |- page.tsx
|- ...
`- package.json
```
