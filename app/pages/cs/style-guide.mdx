## Naming rules

### Format methods

The following are format methods commonly used in programming:

- snake_case
- kebab-case
- MACRO_CASE (SCREAMING_SNAKE_CASE)
- camelCalse
- PascalCase (UpperCamelCase)

### Formats for C# programs

| Item                            | Method      | Example          |
| ------------------------------- | ----------- | ---------------- |
| Namespaces                      | PascalCase  | `MyNamespace`    |
| Classes / Types                 | PascalCase  | `MyClass`        |
| Interfaces                      | IPascalCase | `ISomeInterface` |
| Methods                         | PascalCase  | `MyMethod`       |
| Properties                      | PascalCase  | `SomeValue`      |
| Properties (private, protected) | PascalCase  | `_SomeValue`     |
| Events                          | PascalCase  | `OnEvent`        |
| Fields                          | PascalCase  | `SomeValue`      |
| Enums                           | PascalCase  | `Ok`             |
| Parameter                       | PascalCase  | `someValue`      |

### Order of class values

- Nested classes, enums, delgates, events
- Static, const, readonly fields
- Filed and properties
- Constructor, destructor
- Methods

### Order of elements

- Public
- Internal
- Protected internal
- Protected
- Internal
